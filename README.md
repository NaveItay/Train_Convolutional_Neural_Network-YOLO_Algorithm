This model includes multi-scale identification, some changes in the loss function, and a better feature extraction network.</p><ul id="l7"><li><h1 style="padding-top: 3pt;padding-left: 160pt;text-indent: -18pt;text-align: left;">Network Architecture</h1><p style="text-indent: 0pt;text-align: left;"><br/></p><p style="padding-left: 160pt;text-indent: 0pt;text-align: left;">The
